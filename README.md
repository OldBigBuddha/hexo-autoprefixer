# hexo-autoprefixer

[Autoprefixer] plugin for Hexo.

## Installation

``` bash
$ npm install hexo-autoprefixer --save
```

## Options

``` yaml
autoprefixer:
  exclude: 
    - *.min.js
```

- **exclude**: Exclude files

See <https://github.com/postcss/autoprefixer>

[Autoprefixer]: https://github.com/postcss/autoprefixer
